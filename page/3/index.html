PhotoShop 简明笔记

发表于 2023-04-23 更新于 2023-10-31 本文字数： 291 阅读时长 ≈ 1 分钟

常用快捷键

  • 标尺：Ctrl + R
  • 反选：Ctrl + Shift + I
  • 参考线：从标尺处拖拉可新建参考线；
  • 显示 / 隐藏参考线：Ctrl + ；
  • 内容填充：框选，右键，填充
  • 复制图层：Ctrl + J
  • 将框选的图层复制到新文档：先框选 –> Ctrl + J –> Ctrl + N –> 从剪切板导入。这样做的目的是框选的图层长宽多大，新文档的长宽就是多大，不会有大量多余的空白部分
  • 按比例裁剪：裁剪前可设置裁剪的长宽比（Visio 中似乎没有这个功能，就只能在 PS 中操作了）
  • 对图形进行变换操作：Ctrl + T
  • 对图形进行连续相同的变换操作：Ctrl + Shift + T
  • 取消选区：Ctrl + D
  • 恢复选区：Ctrl + Shift + D
